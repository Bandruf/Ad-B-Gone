Ad-B-Gone

Ad-B-Gone: The simple bookmarklet that removes ads from web pages!

This is a bookmarklet that can block ads for you if you do not have access to extensions! This bookmarklet is very simple to use if you know what you are doing!

  1. Go to https://raw.githubusercontent.com/3kh0/Ad-B-Gone/main/bookmarklet.js

  1. Copy the ALL of the code

  1. Add a new page on your bookmark bar

  1. Paste the code into the page URL and type in ABG into the Name

  1. And thats all! When you want to block the ads just click on the bookmark on the website!

I hope this helps!
